# ANTLRSharp
Calculadora basica implementada en C# junto con ANTLR4.

Utilizar el comando **dotnet run** y modificar el archivo [input.txt](src/InputFiles/input.txt) para probar la calculadora

## Herramientas

- _.NET 8.0_
- _ANTLR 4.13.1_
- _Linux Mint 21.2_
- _VS Code_
